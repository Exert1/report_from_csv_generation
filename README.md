# Анализ макроэкономических данных

## Описание
Программа для составления очтёта по данным из csv файлов в терминале в табличном виде.

## Запуск программы
Для запуска необходимо в терминале вызвать программу через python с обязательными аргументами --files и --report
- --files - Список путей до csv файлов. Разделяется пробелами, без запятых
- --report - Название отчёта. На данный момент присутствует только один отчёт average-gdp. Новый тип отчёта может быть добавлен через создание новой функции с необходимой логикой.

Форма запуска программы через терминвал:

```
python report_generator.py --files <путь до файла1> [<путь до файла2> <путь до файла3> ...] --report <название отчёта>
```


## Пример запуска
```
python report_generator.py --files economics/economic1.csv economics/economic2.csv --report average-gdp
```
